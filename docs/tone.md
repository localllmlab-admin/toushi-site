# トーン&マナー定義（デザイン・文体の単一基準）

対象読者: 20〜40代の日本人男性。仕事の合間・帰宅後に、スマホ/PCで投資を体系的に学び直したい層。
サイトの仕事: 「ここは煽らない、本物の学習サイトだ」と3秒で伝え、学習マップへ導く。
根拠: docs/research/design-refs-*.md（教育・キャリア・メディア11サイトの分析統合）。

## デザインコンセプト —「昼の紙面に、夜の板」

- サイト全体は**白基調の紙面**（11サイト共通の白背景＋抑制配色に従う）。
- ヒーロー・フッターのみ**墨紺の反転面**（トレーダーの板情報のメタファー）。コントラストが個性。
- 影は最小限、罫線（1px border）で構造を作る。新聞・板情報の密度感。

## 信頼の単位（実績数字を持たない新規サイトの代替）

他サイトは「受講者100万人」等の数字で信頼を作る（GLOBIS/スタディング/gacco共通）。
当サイトの信頼の単位は以下。デザインで常時可視化する:

1. **出典等級バッジ**（A=公的機関/B=学術/C=一次発表/D=報道）— 等幅フォントで表示
2. **7章×4レベルの体系**（学習マップ）— 体系性の可視化はGLOBIS/スタディングの転用
3. **レビュー日・更新日**の明示
4. **三原則**: 「煽らない・断定しない・一次情報」— ヒーローに常掲

## 配色運用ルール

- 基調は墨紺（#0b1f33）＋白。アクセント藍青（#1c5bbf）はリンクと主要導線のみ。
- 朱（#c73e2e）は「陽線の朱」。小さな強調マーカー限定。CTA・煽り用途に使わない。
- セクション4色（学ぶ=藍/手法=深緑/チャート=琥珀/用語集=菫）は
  **カード上辺罫線とラベルのみ**（SmartNewsのジャンル色分けの転用）。面で塗らない。

## タイポグラフィ

- 日本語WebフォントはCLS悪化のため不使用。OS搭載スタックのみ（tokens.css参照）。
- 本文17px・行間1.85。見出しはウェイト差（700〜800）で階層を作る（メディア3サイト共通）。
- **等幅フォントは「データの声」**: 日付・出典等級・レベル表記・数値に限定使用。
  本文に使わない。これが金融サイトとしての個性の柱。

## 文体（既存リンター・コンプラ0章と整合）

- 本文は敬体。キャッチコピーのみ常体・体言止め可（R25の距離感）。
- 「同僚の頼れる先輩」の距離感。上から教えない、煽らない、売り込まない。
- 断定禁止・最上級表現禁止（「必ず」「絶対」「No.1」）。条件付き推奨のみ。
- 具体性は数字と固有名詞で出す。ただし出典（等級A/B/C）が付くものだけ。

## 品質床（出荷条件）

- Lighthouse Performance 90以上・CLS 0.1以下
- モバイル対応（カードは縦積み・タップ44px以上）
- キーボードフォーカス可視・prefers-reduced-motion 対応
- JSアイランド追加は最小限（検索ページ以外は原則ゼロJS）
