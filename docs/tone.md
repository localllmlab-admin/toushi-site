# トーン&マナー定義（デザイン・文体の単一基準）

対象読者: 20〜40代の日本人男性。仕事の合間・帰宅後に、スマホ/PCで投資を体系的に学び直したい層。
サイトの仕事: 「ここは煽らない、本物の学習サイトだ」と3秒で伝え、自分に合う記事へ最短で導く。
根拠: design/samples/ の5案比較（A=Yahoo!ポータル/B=NewsPicks/C=Bloomberg新聞/D=転職サイト/E=A×D合成）から
ユーザーがE案を選定（2026-07-07確定）。旧v2「昼の紙面に、夜の板」は置換済み。

## デザインコンセプト v3 —「ニュースポータルの骨格 × 求人ポータルの手ざわり」

- **骨格はニュースポータル**（Yahoo!/ITmedia型）: タブナビ・注目トピックス・カテゴリ別記事リスト・右レール。
  日本人ビジネスマンが毎日見ている画面構造で、既視感＝安心感を作る。
- **手ざわりは求人ポータル**（エン転職/doda型）: 明るいブルー×オレンジ、薄青グレー地に白カード、
  多色バッジ、「条件で探す」検索カード。**読者を「求職者のように条件で記事を探す人」として扱う**。
- 影は控えめ（カード浮きの1段のみ）、ヒーロー検索カードだけ強い影で主役化。

## 信頼の単位（実績数字を持たない新規サイトの代替）

1. **出典等級バッジ**（A=公的機関/B=学術/C=一次発表/D=報道）— 記事リストにも常時表示
2. **7章×4レベルの体系**（学習マップ）— ヒーロー統計「7章×4レベル」とレールで常時可視化
3. **レビュー日・更新日**の明示 — トピックス欄の「◯/◯ 更新」表記
4. **公開本数の正直な表示** — 「4本公開中・60本+執筆予定」。水増ししない
5. バッジは frontmatter の実データからのみ生成する（根拠のない「人気」「おすすめ」表示は作らない）

## 配色運用ルール（tokens.css v3 が正）

- ブランド青のグラデ `#004f86 → #0075c2 → #00a5e3`（--grad-brand）: ヒーロー・レール見出し・ロゴマークのみ。
- オレンジ `#ff6e28`（--color-cta）: **検索ボタン・強アクション・アクティブタブ下線のみ**。
  面で塗らない。売り込みには使わない（アフィリ導入時も PR 表記とセットでのみ）。
- ページ地は薄青グレー `#f2f5f9`（--color-page）+ 白カード（角丸8px・1px罫線）。トップ・一覧で使用。
  記事ページは白地のまま（読み物は紙面）。
- セクション4色（学ぶ=藍#2b4a8b/手法=深緑#1f6e50/チャート=琥珀#a3690f/用語集=菫#6b4fa0）は
  カテゴリラベル・サムネ地・セクション見出しバーのみ。面で塗らない。
- 多色バッジ（緑=読みやすさ/青=出典/紫=レベル/橙=図解）は tokens.css の --badge-* を使う。意味を固定し流用しない。
- 朱（#c73e2e）は「陽線の朱」。チャート図解専用に格下げ。

## タイポグラフィ

- 日本語WebフォントはCLS悪化のため不使用。OS搭載スタックのみ（tokens.css参照）。
- 記事本文16px・行間1.8（16px未満にしない）。トップのリスト・メタ情報は12〜15px相当まで可（ポータルの密度）。
- 見出しはウェイト800、タイトルリンクは太字黒→hoverで青。ニュースサイトの文法に従う。
- **等幅/欧文フォントは「データの声」**: 日付・出典等級・統計数字（--font-num）に限定使用。本文に使わない。

## 文体（既存リンター・コンプラ0章と整合）

- 本文は敬体。キャッチコピーのみ常体・体言止め可。
- ヒーローコピーは「あなたに合う記事から、投資の学習をはじめよう」——学習の入口としての声。煽らない。
- 断定禁止・最上級表現禁止（「必ず」「絶対」「No.1」）。条件付き推奨のみ。
- 具体性は数字と固有名詞で出す。ただし出典（等級A/B/C）が付くものだけ。
- 「人気の検索」等の見出しはナビゲーションの慣用表現として可。ただし記事本文では根拠なき「人気」を使わない。

## 品質床（出荷条件）

- Lighthouse Performance 90以上・CLS 0.1以下
- モバイル対応（検索カード・カード類は縦積み・タップ44px以上）
- キーボードフォーカス可視・prefers-reduced-motion 対応
- JSアイランドは最小限（検索ページ・ヒーロー検索の遷移・/learn/のlvフィルタの3点のみ。追加は要検討）
