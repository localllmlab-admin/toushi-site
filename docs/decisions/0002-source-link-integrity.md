# ADR-0002: 出典リンクの実在性原則と等級E（当サイト編集部調べ）の導入

- 日付: 2026-07-10
- ステータス: 承認（Masaru指示による全面改修）

## 背景

全196記事の出典・参考欄353エントリを監査した結果、約94エントリが出版社トップページ（パンローリング87件・日経BP等7件）、約240エントリが公的機関のドメイントップページ（JPX/日銀/金融庁/JSDA等）へのリンクだった。タイトルは「日本取引所グループ（投資部門別売買状況）」のように具体的資料名を謳うのにリンク先に該当コンテンツが存在せず、Masaruから次の指摘を受けた。

1. リンク先に記事がない出典は根拠にならず、逆に書籍の文章の無断利用を疑わせる
2. パンローリングへのリンクはアフィリエイトでもなく当サイトに収益メリットがない
3. 出典リンクが同一タブ遷移でサイト離脱になる
4. 無理に出典をこじつけない。実記事がなければ「当サイト編集部調べ」等とする
5. 今後、書籍を出典にする場合はAmazonアフィリエイトリンクが利用できるものに限定

## 選択肢

1. 書籍出典をリンクなしgrade Bのまま残す — 書名は残るが上記1の疑念が解消しない
2. 実在ページへの深いリンクに差し替え、なければ等級E（編集部調べ・リンクなし）へ置換 — 採用
3. 全出典をE化 — 公的機関の実在ページまで失われ根拠力が過剰に低下

## 決定

案2を採用。実装は以下。

- スキーマ: `sources[].url`をoptional化・gradeに`E`（当サイト編集部調べ）追加（src/content/config.ts）
- バリデータ: 「A/B/C最低1件、なければE最低1件」に変更（ops/validate_content.mjs）
- 表示: 出典リンクを`target="_blank" rel="noopener noreferrer nofollow"`化・Eはリンクなし表示（src/layouts/Article.astro）
- 全記事修正: 書籍出典94件→E化、公的機関トップページ→WebFetch実在確認済みの深いリンクへ234件差し替え（例: JPX→なるほど！東証経済教室/制度個別ページ、JSDA投資の時間→J-FLECアーカイブ（2024年7月閉鎖・移管を確認）、投資信託協会→資産運用業協会imaj.or.jpへ統合移転を確認）
- 汎用教育コンテンツ（東証経済教室・投資の時間）はテクニカル分析・手法・格言・心理系83記事から除去（準こじつけ回避）。基礎・制度系learn記事のみ保持
- check_links.mjs: ブラウザUA+curlフォールバック・bot全面遮断サイト（JSTOR/doi.org）は死活対象外に
- 編集方針ページ（/policy/）とCLAUDE.md v5に恒久ルール化

## 根拠

- 全差し替えURLはWebFetch/curlで200と内容一致を実確認（確認日2026-07-10・サブエージェント4体で並列検証、独立監査エージェントがサンプル再検証でPASS）
- E化は「出典の等級を下げる」のではなく「実在しない根拠の偽装をやめる」ための正直化。E記事もvalidateで機械検査される
